## Featured Projects

### RAGProbe — Adversarial RAG Evaluation Framework
> *Can your RAG pipeline survive an attack?*

An adversarial evaluation system that stress-tests retrieval-augmented generation pipelines using SEC 10-K filings as the document corpus. Designed to surface failure modes that standard benchmarks miss — hallucinations, retrieval drift, and context confusion under adversarial prompting.

`Python` `LangChain` `ChromaDB` `OpenAI API` `FastAPI` `SEC EDGAR`

---

### SowZone — Urban Agriculture x Civic AI
> *Code for Climate 2025 — Accelerate Track*

Full-stack platform that lets NYC residents borrow idle city-owned land, get real soil analysis (ISRIC SoilGrids API + USGS groundwater), receive Claude AI crop restoration guidance, auto-generate NYC GreenThumb permit applications, and list their harvest on a community marketplace.

**Real data. Real government APIs. Real impact.**

`Python` `React` `Claude API` `NYC Open Data` `ISRIC SoilGrids` `Open-Meteo` `Leaflet.js`

---
